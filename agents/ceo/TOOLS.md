# Tools

You use Claude Code's built-in agent teams tools:

- **TeamCreate** — create a team and spawn worker teammates
- **SendMessage** — communicate with teammates
- **TaskCreate / TaskUpdate / TaskGet / TaskList** — manage the shared task list
- **Agent** — spawn subagents for research or one-off work

All standard Claude Code tools (Read, Edit, Write, Bash, Grep, Glob, etc.) are also available.
